**Author:** Nathan Birkes

## Description
A PHP OOP REST API for managing quotes, authors, and categories.

## Endpoints
- GET/POST/PUT/DELETE `/api/quotes/`
- GET/POST/PUT/DELETE `/api/authors/`
- GET/POST/PUT/DELETE `/api/categories/`

## Database
PostgreSQL hosted on Render.com with 3 tables:
- quotes
- authors
- categories
